# Trame noire (Темна тканина)

> Концепція мережі просторів з обмеженим штучним освітленням, що забезпечує екологічну зв'язність для нічних видів.

## Походження концепції

Термін `trame noire` сформувався у Франції приблизно у 2010-х роках як логічне розширення раніше існуючої концепції **trame verte et bleue** — зеленої й блакитної тканини (зелені простори + водні шляхи), що становлять базис французької національної екологічної мережі (`Stratégie nationale Trame verte et bleue`, 2014).

Trame noire додає **нічний вимір** до екологічної зв'язності: визнається, що нічні і присмеркові види (комахи, кажани, нічні птахи) потребують неосвітлених коридорів для переміщення, як рослини потребують зелених коридорів.

## Принцип роботи

```
        ДЕНЬ                            НІЧ
   ┌─────────────┐                ┌─────────────┐
   │ trame verte │                │ trame noire │
   │   parcs     │                │   couloirs  │
   │   bois      │                │   sombres   │
   └─────────────┘                └─────────────┘
       Підтримка                   Підтримка
       денних видів                 нічних видів
```

## Реалізація у досліджуваних містах

### Ліон — метрополітальна шкала
Trame noire визначено на рівні Métropole de Lyon. SIGERLy у партнерстві з Ville de Lyon і колегами працює над:
- Виявленням зон з підвищеною нічною біорізноманіттю
- Картуванням маршрутів кажанів
- Узгодженням освітлення з біокоридорами

→ [[cities/Lyon_Логіка_рішень]]

### Женева — центральна структура плану
Це найбільш формалізована імплементація trame noire серед досліджуваних міст. План 2021 побудовано **навколо** темної тканини:

| Зона | Освітлення |
|---|---|
| `noires` (берег озера, Парк Бертран) | мінімум або тільки на детекцію 8вечора-6ранку |
| `sombres` (наприклад Парк Гайзендорф) | теплі низькі рівні, зниження після 22:00 |
| `urbaines` | звичайне, з вимкненням ілюмінацій опівночі |

→ [[cities/Geneva_Логіка_рішень]]

### Копенгаген — наукова підстава
Біологічна студія, замовлена муніципалітетом для `Belysningsmasterplan del 2`, визначила вразливі зони. Це не оформлено як `trame noire` офіційно, але реалізує ту саму ідею.

## Технічні рішення для trame noire

- **Geographic placement** — освітлення розташовується далі від чутливих зон
- **Directional optics** — повний cut-off для запобігання потрапляння світла на дерева, водойми
- **Adaptive dimming** — автоматичне зниження або вимикання вночі за відсутності користувачів
- **Spectral selection** — теплий спектр (≤2700K), бажано без сильної синьої компоненти
- **Vegetation barriers** — використання рослинності як "світлового шилда"
