# Lichtaudit / Lighting audit

> Систематичне обстеження міського освітлення — інвентаризація, оцінка стану, виявлення дефіцитів і проблем.

## Призначення

Lighting audit — **обов'язкова перша фаза** будь-якого світлового майстер-плану. Без точної картини "як зараз" неможливо ефективно проєктувати "як має бути".

## Етапи аудиту

### 1. Інвентаризація обладнання
- Тип світильника (HPS, металогалогенні, LED)
- Потужність
- Висота встановлення
- Тип опори (стовп, кронштейн, ferro)
- Рік встановлення / останнього сервісу
- Стан / придатність

**Інструмент**: GIS-база даних із унікальним ID для кожної світлоточки.

### 2. Світлотехнічні вимірювання
- **Ē** (середня горизонтальна освітленість)
- **Uo** (загальна рівномірність)
- **L** (яскравість покриття для M-класів)
- **CCT, CRI** (кольоровий аналіз)
- **Skyglow** (вимірювання у напрямку зеніту)

**Інструмент**: Люксметр, спектрометр, портативна камера з калібровкою.

### 3. Нічні обстеження
- Walking surveys в темну пору
- Дронове обстеження для великих територій
- Мапування зон з:
  - надмірним освітленням
  - light trespass
  - dead zones (нестача)
  - пожежонебезпечним станом

### 4. Сприйняттєвий аудит
- Опитування мешканців
- Карти "де комфортно/некомфортно вночі"
- Гендерний аналіз сприйняття

### 5. Біологічна оцінка
- Огляд біорізноманіття вздовж вулиць
- Проби на летучих мишів
- Аналіз впливу на birds (особливо мігрантів)
- Перетин з охоронюваними територіями

### 6. Енергетичний аналіз
- Поточне споживання (кВт·год/рік)
- Витрати
- Емісія CO2
- Базлайн для порівняння after-renovation

## Спеціалізовані інструменти

### Метрики skyglow
- **VIIRS** супутникові дані (NASA)
- **Light at Night** проекти від ESA
- **Falchi World Atlas** — глобальний показник
- Локальні sky quality meters (Unihedron SQM-L)

### Insect monitoring
- Світлові пастки (light traps) з фіксацією
- Time-lapse камери для оцінки insects swarm
- Моніторингові маршрути

### Bat monitoring
- Bat detectors (ANABAT, EchoMeter)
- Картування echolocation calls
- Коридори літання

## Кейси

### KADK Mapping of Copenhagen (2014)
**Det Kongelige Akademi для Архітектури, Дизайну і Консервації** виконав комплексний картоопис вуличного освітлення Копенгагена як підставу для Belysningsmasterplan. Висновки публіковано через issuu.com — надзвичайно ціний відкритий референс.

### Vienna catalog 2008/2011
MA 33 + MA 19 разом з podpod створили **lighting catalog** з типологією для Відня. У 2011 — оновлено зі стандартизованою "Виденською сім'єю" ламп-постів.

→ [[cities/Vienna_Процес_створення]]

### Lyon — energy baseline
Аудит у 2000 встановив базу: **42 GWh/рік**. Після LED-ery і dimming-програм — **21 GWh/рік** у 2021. Це дозволило кількісно довести успіх Plan Lumière 2.

## Lichtaudit як живий процес

Сучасний підхід — **continuous auditing** через:
- Real-time моніторинг через CMS ([[concepts/Smart_lighting_та_контроль]])
- Періодичні (раз у 5 років) глибокі ревізії
- Інтеграція з citizen complaints системою

Це робить Lichtaudit не одноразовою подією, а постійним режимом міста.
