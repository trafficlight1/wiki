# Мадрид — огляд

> Дві паралельні стратегії: **Plan Director de Iluminación** для UNESCO-кластера Прадо/Буен Ретіро + **Madrid Renovable y Eficiente** для масштабованої LED-конверсії.

## Ключові факти

| Параметр | Значення |
|---|---|
| UNESCO Plan Director | **Paseo del Prado y Buen Retiro** — Plan Director de Iluminación |
| Бюджет та контекст | UNESCO heritage Management Plan 2021 |
| LED стратегія | **Madrid Renovable y Eficiente** |
| Перший великий проект LED | 2014 з Philips/Signify (225,000 точок) |
| Замовник | Ayuntamiento de Madrid |
| Кількість світлоточок (full city) | **~253,575** |
| LED conversion (станом на 2025) | **66.9%** (від 35.6% у 2019) |
| Цільова дата 100% LED | **2027** (через Renovable y Eficiente) |
| Інвестиція в обслуговування | 76 млн EUR (2024 contract extension) |
| Member | UNESCO World Heritage Cities |

## Унікальність Мадрида

### UNESCO-driven підхід
Включення `Paseo del Prado y Buen Retiro, paisaje de las artes y las ciencias` у World Heritage List **липня 2021** створило **обов'язкове управлінське зобов'язання**. Plan Director de Iluminación розроблено саме для виконання вимог Heritage Management Plan.

### Найбільший світовий проект LED-конверсії (2014)
**225,000** світильників замінено на Philips connected LED — на час контракту це був **найбільший** проект свого роду в світі.

### Бізнес-модель — ESCO через тендер
Поділ на 3 лоти (Centre, West, East):
- LUZ MADRID consortium — лоти 1+2
- INMAES consortium — лот 3

Це дозволило місту:
- **Без капітальних витрат** виконати конверсію
- ESCO компанії повертають інвестицію через зекономлені кошти
- Citizens не платять додатково

### Plan Director як модель для центру
План для Прадо/Буен Ретіро задумано як **прецедент** — модель для подальшого розширення на історичний центр Мадрида.
