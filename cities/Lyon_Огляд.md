# Ліон — огляд

> Перше місто, що сформулювало термін `Plan Lumière` (1989). Поточний документ — **3-й Plan Lumière**, ухвалений 11 травня 2023.

## Ключові факти

| Параметр | Значення |
|---|---|
| Поточний документ | **3-й Plan Lumière** (Le Plan Lumière : prendre soin de la nuit) |
| Ухвалено | 11 травня 2023, Conseil municipal |
| Розробник | Urbalyon (Urban Planning Agency) + Direction de l'Éclairage Urbain |
| Концертація | 2019-2023 |
| Перший Plan Lumière | 1989 |
| Другий Plan Lumière | 2004 |
| Кількість світлоточок | ~75 000 |
| Енергоспоживання | 21 GWh (2021), -50% від 2000 |
| Член | LUCI Association (засновник, 2002) |

## Суть документа

3-й Plan Lumière побудовано на трьох опорних цінностях:

```
        ┌─────────────────────────────────┐
        │      PLAN LUMIÈRE — 3           │
        │   "prendre soin de la nuit"     │
        └────────────┬────────────────────┘
                     │
        ┌────────────┼────────────────────┐
        │            │                    │
        ▼            ▼                    ▼
   QUALITÉ      SOBRIÉTÉ          CITOYENNETÉ
   (якість)     (стриманість)     (громадянськість)
   
   - сприйняття    - енергія         - концертація
   - атмосфера     - біорізноманіття - персоналізація
   - ідентичність  - sky glow        - локальна адаптація
```

## Унікальність Ліона

### Перший у світі майстер-план
До 1989 більшість міст не мали стратегічного підходу до нічного освітлення. Ліон поставив його у центр urban planning і **створив прецедент** для решти Європи.

### LUCI Association
У 2002 Ліон ініціював `Lighting Urban Community International` — мережу міст, що співпрацюють у сфері якісного освітлення. Серед членів — Копенгаген, Женева, інші досліджувані тут міста.

### Fête des Lumières
Щорічний фестиваль світла перетворив Ліон на лабораторію публічних експериментів зі світлом. Фестиваль — 4 дні в грудні, понад 2 мільйони відвідувачів.

### Метропольна шкала trame noire
Ліон розширив свій план до Métropole de Lyon (59 комун) через:
- партнерство з SIGERLy
- координацію з Plan Climat Air Énergie Territorial
- метропольну `[[concepts/Trame_noire|trame noire]]`
