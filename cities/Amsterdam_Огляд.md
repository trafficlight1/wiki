# Амстердам — огляд

> Унікальний підхід — замість майстер-плану **політична рамка** (`Beleidskader Verlichting`). Інтегрує приватне і публічне освітлення в єдиний підхід.

## Ключові факти

| Параметр | Значення |
|---|---|
| Поточний документ | **Beleidskader Verlichting** |
| Ухвалено | 2017 (комісія Infrastructure & Sustainability, 1.06.2017) |
| Громадська консультація | кінець 2016 |
| Дизайн-партнер | **DSGN.FRM** (інфографіка), Rolf Uhlhorn (концепт) |
| Замовник | Gemeente Amsterdam |
| Кількість світлоточок | ~115 000 |
| Інтерактивна карта | maps.amsterdam.nl/lichtpunten/ (публічна) |
| Інший документ | `Beleidsplan openbare verlichting 2018-2027` (operational) |

## Концепція "Beleidskader" — політична рамка, не план

```
ТРАДИЦІЙНИЙ                       АМСТЕРДАМСЬКИЙ
МАЙСТЕР-ПЛАН                      BELEIDSKADER
                                  
Зональність ──┐         Принципи ──┐
Норми       ├─►        Завдання  ├─► (інтерпретується
Графіки     ─┘         Параметри ─┘   локально)
                                  
"що робити             "як думати
конкретно"             про світло"
```

Це дозволяє:
- адаптацію до різних кварталів
- швидке оновлення без перегляду документа
- залучення приватних акторів через ту саму рамку

## Інтегрований підхід

Унікальна риса Амстердама — **єдина рамка для приватного і публічного** освітлення:
- Гемента відповідає за публічне (вуличне)
- Та сама рамка задає очікування для приватного (фасади, реклама, internal)
- Прозорість: інтерактивна карта `maps.amsterdam.nl/lichtpunten/` показує всі публічні світлоточки

## Пріоритети

З документа:
> *Verlichting draagt bij aan de leefbaarheid in de stad. Sociale veiligheid, verkeersveiligheid en sfeer in de openbare ruimte zijn gebaat bij goede verlichting.*

Тобто:
1. **Соціальна безпека**
2. **Дорожня безпека**
3. **Атмосфера в публічному просторі**
4. **Стабільність обслуговування за розумної вартості**
5. **Sustainability** — світло не горить без потреби

## Унікальність Амстердама

### Beleidskader як формат
Не майстер-план з конкретною зональністю, а **політична рамка**, що задає принципи інтерпретації для конкретних рішень. Це гнучкіше, але вимагає більш кваліфікованих локальних рішень.

### Інтерактивна публічна карта
**`maps.amsterdam.nl/lichtpunten/`** — кожна публічна світлоточка з ID, типом, статусом. Це вищий рівень прозорості, ніж у будь-якого з інших досліджуваних міст.

### Інтеграція з ширшим документом
**Beleidsplan openbare verlichting 2018-2027** — operational, доповнює стратегічну рамку. Дві документи разом дають повну картину.

### Visie Openbare Ruimte 2025
Освітлення інтегровано з ширшою стратегією публічного простору ("De huiskamer van alle Amsterdammers" — гостинна для всіх амстердамців).
