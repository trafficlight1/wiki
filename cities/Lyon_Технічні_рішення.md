# Ліон — технічні рішення

> Світлотехнічний аналіз, технології, конкретні параметри і виміри впливу 3-го Plan Lumière.

## Світлотехнічний аналіз

### Базові параметри (за EN 13201)

Для основних типів дороги в Ліоні:

| Тип | Клас EN 13201 | Ē / L (ціль) | Uo |
|---|---|---|---|
| Магістралі | M3 | L = 1.0 cd/m² | 0.4 |
| Колектори | M4-M5 | L = 0.5-0.75 cd/m² | 0.4-0.35 |
| Розв'язки | C2-C3 | Ē = 15-20 lx | 0.4 |
| Жилі | P3-P5 | Ē = 3-7.5 lx | 0.4 |
| Велозон / пішохідні | P5-P6 | Ē = 2-3 lx | 0.4 |

### Кольорова температура

Жорстка ієрархія за зонами:

```
2200K  ──── trame noire, природні зони
       │
2700K  ──── житлові, тихі квартали
       │
3000K  ──── активні вулиці (макс.)
       │
       └── Заборонено вище 3000K
```

### Glare control
- **Full cut-off** оптика (UL = 0%) для нових установок
- TI (threshold increment) ≤15 для пишохідно-вуличних класів
- Diffusers для пишохідних ділянок

## Технології

### LED — домінуюча технологія
- Поступовий перехід з натрієвих ламп з 2000-х
- 50%+ парку — LED (станом на 2021)
- Мета — 100% LED до 2030

### Smart-керування
- **Detection-based dimming** — PIR/радар у низькозалюднених зонах
- **Bi-level scheduling** — два режими (повний / зниженний)
- **Centralized monitoring** — CMS-платформа

### Архітектурне освітлення
Партнерство з:
- **Light designers** з різних бюро
- **Conservation des Monuments Historiques** для спадщини
- Спеціальні проекти для **Fête des Lumières**

→ Деталі: [[concepts/Smart_lighting_та_контроль]]

## Геодослідження та просторовий аналіз

### GIS-інтеграція
- База даних усіх 75 000 світлоточок
- Шари: світло, біорізноманіття, спадщина, скарги мешканців
- Інтеграція з **Métropole de Lyon GIS**

### Trame noire mapping
- Визначення біокоридорів через SIGERLy
- Особливий фокус — кажани (chiroptères)
- Мапування потенційних проблем (мосту над Роною як приклад)

### Skyglow моніторинг
- Регулярні нічні спостереження з міста і пагорбів
- Партнерство з астрономічними клубами

## Виміри впливу

### Енергоспоживання

```
2000  ────  42 GWh/рік
            ↓ (LED + dimming + smart)
2021  ────  21 GWh/рік   (-50%)
            ↓ (3-й Plan Lumière)
2030  ────  ~14-16 GWh (ціль)   (-65-70%)
            ↓
2050  ────  ~10 GWh (ціль)      (поділ на 4)
```

### CO2-емісія
Зменшення пропорційне споживанню — **-50%** з 2000 по 2021. Конкретна цифра залежить від вуглецевої інтенсивності французького електропостачання (низька — 60-80 g CO2/kWh).

### Біорізноманіття
- Експерименти з 2200K у деяких локаціях показали **значне зменшення** комах біля світлоточок
- Картування кажанів встановило коридори для уникнення

### Безпека
Метастудії показують, що **зменшення освітлення** в Ліоні **не призвело до зростання злочинності**. Це доводить, що sobriété можлива без втрати безпеки.

### Якість сприйняття
Опитування мешканців після пілотних проектів:
- **+30%** оцінювали "затишність" вище
- **+15%** оцінювали "якість сну" вище
- **-10%** скаржилися на "заіграву"

## Конкретні проекти-приклади

### Place Bellecour
- Зменшення інтенсивності
- Зміна спектра на тепліший
- Збереження розпізнаваного нічного силуету

### Берег Рони
- Імплементація trame noire
- Збереження мінімального освітлення для пішохідного руху
- Захист коридору кажанів

### Vieux Lyon (UNESCO World Heritage)
- Архітектурна ілюмінація — погасання опівночі
- Тепле світло (2700K)
- Мінімізація light trespass
