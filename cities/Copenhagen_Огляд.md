# Копенгаген — огляд

> Belysningsmasterplan з біологічною і соціологічною студіями як основою. Наймасштабніший серед досліджуваних інфраструктурний проект.

## Ключові факти

| Параметр | Значення |
|---|---|
| Поточний документ | **Belysningsmasterplan** (Parts I + II) |
| Затверджено | 24 лютого 2014, Teknik- og Miljøudvalg |
| Бюджет | **261,3 млн DKK** (рішення 12.12.2013) |
| Партнер реалізації | **Citelum** (EDF Group) |
| Розробник аналітики | **KADK** (Royal Danish Academy) |
| Кількість світлоточок | ~38 000 |
| LED-конверсія | 18 800 точок (2014-2017) |
| Ширша рамка | CO2-нейтральність 2025 |
| Член | LUCI Association |

## Структура документа

Belysningsmasterplan складається з двох частин:

```
PART I (виконано 2014-2015)
   - Опис проекту renewal
   - Технічна основа LED-конверсії
   - Зональна організація для всього міста
                    ↓
PART II
   - Майбутнє освітлення з input громади
   - Wishes від local councils
   - Біологічна студія
   - Соціологічна студія
```

## Унікальність Копенгагена

### Біологічна + соціологічна студія як обов'язкова основа
Унікально — Belysningsmasterplan part 2 базується на:
- **Biological study** — як освітлення впливає на дику природу міста
- **Sociological analysis** — як копенгагенці сприймають освітлення

Це **єдиний** серед досліджуваних кейс, де природничі і гуманітарні науки **формально** інкорпоровані як підставовий шар.

### Найбільший інфраструктурний проект
261,3 млн DKK (~35 млн EUR) на single project — це найбільша світлова інвестиція serед досліджуваних міст у відносному масштабі.

### KADK Mapping
**Det Kongelige Akademi для Архітектури, Дизайну і Консервації** виконав **`Mapping of Copenhagen Street Lighting 2014`** — публічний відкритий ресурс, що залишається унікальним прикладом academic engagement з муніципальною темою.

→ [issuu.com/kadk/docs/mapping_of_copenhagen_street_lighti](https://issuu.com/kadk/docs/mapping_of_copenhagen_street_lighti)

### Citelum partnership
Citelum (EDF Group, Франція) — впровадженний партнер. Через **ESCO/contract**-модель Citelum:
- розробив **panneau SDAL** (графічний огляд)
- встановив 18,800 LED + 8,000 ламп-постів
- здійснює довгострокову експлуатацію

### Local councils integration
Belysningsmasterplan part 2 явно інтегрує:
- Бажання районних рад (`lokaludvalg`)
- Локальні пропозиції щодо нових світлоточок
- Інтерпретацію the city through residents

## Ключові цілі

1. **Покращення безпеки** і комфорту мешканців
2. **CO2-нейтральність 2025** — освітлення як один з кластерів
3. **Енергозбереження** — мета >50%
4. **Захист спадщини** і чутливих зон
5. **Зменшення light pollution**
