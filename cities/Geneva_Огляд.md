# Женева — огляд

> Місто, що зробило `trame noire` (темну тканину) **центральним** концептом світлового планування. Plan Lumière 2009, переглянутий 2021.

## Ключові факти

| Параметр | Значення |
|---|---|
| Поточний документ | **Plan Lumière** (revised) |
| Перша редакція | 2009 |
| Переглянута | 2021 |
| Розробник перегляду | **Radiance 35** (lighting design firm) |
| Замовник | Ville de Genève |
| Кількість світлоточок | ~25 000 |
| Структура документа | Plan directeur lumière + еволюційна Charte |
| Член | LUCI Association |

## Ключова інновація

**Trame noire** як центральна структура плану — Женева стала **першим містом Європи**, де темна тканина не додаток, а **організуючий принцип**.

```
                ┌──────────────────────┐
                │   PLAN LUMIÈRE       │
                │   GENÈVE 2021        │
                └──────────┬───────────┘
                           │
                           ▼
                ┌──────────────────────┐
                │  TRAME NOIRE         │
                │  визначає простір    │
                │  ↓                   │
                │  потім розставляєть- │
                │  ся освітлення       │
                └──────────────────────┘
```

## Триступенева зональність

Перегляд 2021 поділив місто на **три зони** з різним нічним характером:

| Зона | Опис | Освітлення |
|---|---|---|
| **Noires** (чорні) | біокоридори, набережна, парки | мінімум або тільки на детекцію 20:00-06:00 |
| **Sombres** (темні) | парки з основними шляхами | теплі низькі рівні, dim після 22:00 |
| **Urbaines** (міські) | центральні вулиці | стандартне з вимиканням ілюмінацій опівночі |

## Унікальність Женеви

### Trame noire як основа, не доповнення
На відміну від Ліона, де trame noire — додатковий шар до традиційного Plan Lumière, у Женеві **темна тканина визначає** все інше. Спочатку малюється карта чорних і темних зон, потім — освітлення в урбаністичних.

### Двочастинна структура документа
- **Plan directeur lumière** (фіксований) — стратегічні осі і теми
- **Charte** (еволюційна) — технічні рекомендації для впровадження

Це дозволяє оновлювати технічні деталі без перегляду стратегії.

### Радіація-підхід Radiance 35
Бюро **Radiance 35** (Genève) реалізувало Plan через "лучний" підхід — світло поширюється "як радіус", з керованою спадкою інтенсивності, не як точкова інстанція.

### Інтеграція з SIG (Services Industriels de Genève)
SIG — муніципальна утіліта — партнер реалізації, аналог цюрихського ewz.
