# Амстердам — технічні рішення

> Як технічно реалізується Beleidskader. Технології, прозорість через GIS, операційні особливості.

## Світлотехнічний підхід

### Стандарти
- **NEN-EN 13201** — нідерландська імплементація європейського стандарту
- Гарантує сумісність на ринку
- Допускає внутрішні Amsterdam-адаптації

### Класифікація
Конвенціональна за EN 13201:
- M-класи для магістралей
- C для конфліктних
- P для жилих

З тенденцією до **нижчих класів** в житлових для зменшення light trespass.

## Технології

### LED-конверсія
- Поступовий перехід (2017→)
- Без жорсткого графіка
- Економічна модель: through-life cost

### Параметри LED
- **Світлова віддача**: ≥120 lm/W
- **CCT**: переважно 3000K
- **CRI**: ≥70
- **Lifetime**: ≥80 000 годин

### Smart-керування
Конкретно реалізується **по проектах**:
- Bi-level dimming на нових установках
- Detection-based в окремих кварталах
- Centralized monitoring через Gemeente систему

→ [[concepts/Smart_lighting_та_контроль]]

## Геодослідження і просторовий аналіз

### Публічна інтерактивна карта
**`maps.amsterdam.nl/lichtpunten/`** — найкраща публічна карта серед досліджуваних міст.

Доступне:
- **115 000+ світлоточок** з координатами
- ID кожної
- Тип, потужність
- Статус
- Можна зум до окремої

Це:
- Інструмент прозорості
- Платформа для citizen complaints
- Дані для аналізу науковцями

### GIS-інтеграція
- Інтеграція з основною GIS-системою Amsterdam
- Шари: світло, urban planning, mobility, sustainability
- Open data politiсs

### Heritage-зонування
- Спеціальний підхід для UNESCO **grachtengordel** (канальний пояс)
- Координація з Heritage protection
- Тепле, м'яке освітлення для каналів

## Виміри впливу

### Енергоспоживання
Кліматична амбіція - **climate neutral 2050**. LED-конверсія - один з основних інструментів.

### Якість сприйняття
- Опитування мешканців
- Відстеження скарг через 311
- Зворотний зв'язок через Beleidsplan operational

### Біорізноманіття
- Початкова інтеграція в нові проекти
- Особлива увага канало-краєв і паркам
- Менше формалізована, ніж у Женеві / Ліоні

### Прозорість
**Унікальний** ефект — публічна карта створює:
- Громадянський контроль
- Швидке усунення несправностей
- Дані для дослідників

## Конкретні проекти

### Grachtengordel (канальний пояс UNESCO)
- Спеціальний heritage-режим
- Тепле, низьке світло
- Підкреслення архітектурних рис
- Захист водної біоти

### Centrum (центр)
- Активне освітлення з обмеженнями
- Архітектурне підкреслення
- Pedestrian-friendly

### Wijken (житлові)
- Помірне освітлення
- Adaptive dimming у нових проектах
- Інтеграція з Puccinimethode

### Vondelpark і інші парки
- Мінімальне освітлення на основних шляхах
- Тепле, обмежене
- Захист нічних видів

## Партнери постачальники

- **Heijmans** — інженерні послуги
- **Schreder** — ламп-пости (через тендери)
- **Philips/Signify** — точкові
- **Selux, Trilux** — спеціальні

## Особливість прозорості

Інтерактивна карта `lichtpunten` — це не просто маркетинг. Це:
- Підстава для всього управління
- Платформа для участі
- Постійний lichtaudit без додаткового бюджету
- **Modulі для citizen science**
